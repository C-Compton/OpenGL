void CAccelerometer::GetOrientation ( float &  Pitch,
float &  Roll,
float &  Yaw

Retrieves the smoothed device attitude (pitch, roll, and yaw) computed with an exponential moving average.

Parameters:
Pitch[out] Reference variable where the smooth device pitch will be set.
Roll[out] Reference variable where the smooth device roll will be set.
Yaw[out] Reference variable where the smooth device yaw will be set. Please, note that without IR enabled, yaw cannot be retrieved.
